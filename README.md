# GoStyle

Projet visant à permettre la lecture

## Initialisation

Etapes d'initialisation du projet

### Prérequis

* React-native

### Librairies utilisées

* [Laravel 5.8.15](https://laravel.com/) PHP web framework
* [Laravel Voyager](https://laravelvoyager.com/) Admin panel for Laravel
* [Webpack](https://webpack.js.org/) Dependency management

## Déploiement

// ajouter le déploiement

```
React-native start
```

Lancer le construction de l'application pour Android

```
React-native run-android
```
